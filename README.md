## 📋 Descripción del Proyecto

Una invitación digital elegante y responsiva para celebrar los 25 años de matrimonio. Diseño moderno con tema de plata y oro, animaciones suaves y una experiencia de usuario excepcional.

---

## 🎨 Características

✨ **Diseño Elegante**: Tema de plata y oro con colores cálidos
📱 **Responsivo**: Se adapta a cualquier dispositivo (móvil, tablet, desktop)
🎞️ **Sección de Fotos**: Marco decorativo para la foto del primer matrimonio
🌟 **Animaciones**: Transiciones suaves y efectos visuales
🖨️ **Imprimible**: Optimizado para impresión
⌨️ **Accesible**: Navegación por teclado soportada

---

## 🖼️ Cómo Agregar la Foto

1. **Coloca la foto del matrimonio** en la carpeta `src/assets/images/`
2. **Nombra el archivo** como `wedding-photo.jpg` (o actualiza el nombre en `index.html`)
3. **Formatos soportados**: JPG, PNG, WebP

**Recomendaciones de la foto:**
- Resolución mínima: 800x600px
- Relación de aspecto: 4:3 o similar
- Tamaño del archivo: menos de 2MB para mejor rendimiento

## 🚀 Cómo Usar

### Opción 1: Abrir en el navegador
```bash
# Abre el archivo index.html directamente en tu navegador
open index.html
```

### Opción 2: Servidor local (recomendado)
```bash
# Usa Python 3 para servir los archivos
python3 -m http.server 8000

# Luego accede a: http://localhost:8000
```

### Opción 3: Con Live Server (VS Code)
- Instala la extensión "Live Server" en VS Code
- Haz clic derecho en `index.html`
- Selecciona "Open with Live Server"
